# HAPEL Method Parameters

---

## $text
### (null|string), (required)

The text for the tag's inner content. 

| Value  | Use                                                   |
|--------|-------------------------------------------------------|
| string | wraps the contents of 'string' with the tag           |


### Examples:

```php
$text = 'Cannot load javascript.';
```
```html
<tag>Cannot load javascript</tag>
```
