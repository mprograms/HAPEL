# HAPEL Method Parameters

---

## $src
### (string), (required)

Adds the src attribute to an HTML tag.

Value      | Use
-----------|-------------
string     | adds the value of 'string' as the attribute


### Examples:

```php
$src = 'myFile.html';
```
```html
<tag src="myFile.html" {...}
```
