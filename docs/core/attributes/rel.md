# HAPEL Method Parameters

---

## $rel
### (string), (required)

Adds the rel attribute to an HTML tag.

Value      | Use
-----------|-------------
string     | adds the value of 'string' as the attribute


### Examples:

```php
$rel = 'shortcut-icon';
```
```html
<tag rel="shortcut-icon" {...}
```
